Methods

(static) conflate(caller, tester, final_callback)

Conflates calls for "paginated" calls, for example Dropbox API returns For more details - see the test. {has_more: true, changes: [...]} if there are more results to poll, and {has_more: false, changes: [...]} if there are no more results.
Parameters:
Name Type Description
caller function function used to make the call, caller(conflate_callback, last_response_arg1, last_response_arg2...) - conflate callback should be used as the main callback for the action
tester function function that should return true if another call should be takes, tester(last_response_arg1, last_response_arg2...)
final_callback function function that will be called after all results are collected, accepts on argument with list of all response arguments passed to the result callback: final_callback([[first_response_arg1, first_response_arg2...],[second_response_arg1, second_response_arg2, ...],...).
