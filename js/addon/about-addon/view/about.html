'afterwriting labs is a place where you can play with some screenwriting tools. You can open screenplays written in Fountain format or Final Draft (it will be converted to fountain). You can also use one of the samples.

You can use it offline too! (sorry, no Dropbox/Google Drive support for offline version). Just download this file, unzip and double click on afterwriting.html.

It's a client-side only app. That means your screenplay is never sent out of your PC.

What's inside?

Available plugins:

  •  /    Open / Save - open a .fountain or .fdx file from disk, Dropbox or use a sample, save as .fountain or as PDF.
  • Settings - change paper size, print layout, etc.
  • Editor - a basic editor for .fountain files
  • Preview - PDF preview (a browser plugin required)
  • Facts - facts about your screenplay (number of pages, scenes, characters, etc.)
  • Useless Stats - some statistics, don't take them too seriously!

Privacy

'afterwriting is a font-end only, open-source application. Your screenplay is not sent out unless you use Dropbox / Google Drive integration (then your screenplay is sent over SSL connection). PDF is generated inside the browser which means you can go offline and still write your screenplay!

The main reason behind it is to make sure the application can be run anytime, anywhere, even if you have no Internet connection. You can download afterwriting.zip file which contains the whole website. I recommend you to do it even if you are "always online" in case the website is down* or you forget to pay the pill.

(*) 'afterwriting is hosted on GitHub Pages which is available 99% of the time. However according to Murphy's law "if a website is available 99% of the time, you will critically need to use during the other 1%", so... get a backup today!
